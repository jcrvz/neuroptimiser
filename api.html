class neuroptimiser.core.processes.TensorContractionLayer(*args, **kwargs)

Bases: AbstractProcess

Tensor Contraction Layer (TCL) This layer is used to contract the input tensor along the specified axes. This particular implementation is based on the einsum function from numpy to compute the tensor contraction:

A_{ij} = sum_{k} W_{ijk} S_{k}

where A_{ij} is the output tensor, W_{ijk} is the weight tensor representing the synaptic weights (adjacency), and S_{k} is the input tensor representing the spikes.
